# 🎬 Movie Recommendation System

This project is a content-based movie recommendation system using Machine Learning.

## 🔍 Features
- Recommends movies based on similarity
- Uses cosine similarity
- Built using Streamlit

## 🧠 Tech Stack
Python, Pandas, Scikit-learn, NLTK, Streamlit

## ⚠️ Note
The .pkl files are not included due to size limitations. 
Run the notebook to generate them.

## ▶️ Run
streamlit run app.py
